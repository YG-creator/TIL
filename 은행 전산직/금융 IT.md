# 1. 지급 결제란?

지급 : 지급 수단을 주는 행위

결제 : 대금 등을 주고 받아 거래관계를 끝맺는 일

![image-20220624214223757](md-images/image-20220624214223757.png)

실제로 인터넷 뱅킹은 지급과 결제가 동시에 이뤄지지 않음

지급을 먼저하고 결제를 나중에 한꺼번에 함 -> 실시간 가능

금융결제원이 전산처리를 하루 동안 한 후 -> 한국은행에 보냄 -> 한국은행이 결제
